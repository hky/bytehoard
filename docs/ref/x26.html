2.2. Common Field Names

The same field names are used all over the place, for simplicity. Here's a reference of them.

  • filepath - A bytehoard-style filepath. See Concepts - Filepath.

  • filetype - The file's extension.

  • username - A username. Duh.

  • password - A (MD5ed) password.

  • group - A group name.

  • status - The status of the other fields linked together. Used in ACLs, module lists.
